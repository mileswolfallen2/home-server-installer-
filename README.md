# Home Server Installer

This repository contains a shell script (`.sh`) to automate the installation of various applications on a home server.

## Description

- **Purpose:** Quickly set up essential applications on your home server using a single script.
- **Language:** Shell script (`.sh`)

## Features

- Installs multiple apps automatically.
- Simple setup: Just run the script.

## Usage

1. **Clone this repository:**
   ```sh
   git clone https://github.com/mileswolfallen2/home-server-installer-.git
   cd home-server-installer-
   ```

2. **Make the script executable:**
   ```sh
   chmod +x install.sh
   ```

3. **Run the installer:**
   ```sh
   ./install.sh
   ```

> **Note:** Review `install.sh` before running to ensure it meets your requirements and is safe for your server.
